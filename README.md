# First-task
Testavimo, 1-oji užduotis - Įrėmintas pasisveikinimas

## Programos veikimas:

Paleidus programa, prašoma įvesti mažiausiai 3 simbolių ilgio vardą:


```shell
Enter your name: 
Tomas
*******************
*                 *
* Sveikas, Tomas! *
*                 *
*******************
```

Jei vartotojo įvestas vardas yra moteriškas, programa prisitaiko pakeisdama žodį:

```shell
Enter your name: 
Smilte
*******************
*                 *
* Sveika, Smiltė! *
*                 *
*******************
```
